# 星际先锋 (Star Craft)

一款视觉效果出色的太空战机网页游戏，使用 React + Tailwind CSS 构建。

## 游戏特色

- 🚀 四种独特战机可供选择，各具特色
- 👾 多种类型敌机（基础型、快速型、重型）
- 🎁 道具系统（三向子弹、能量护盾）
- 🏆 成就系统
- 📱 响应式设计，支持手机、平板、电脑

## 战机介绍

### 蓝隼-Alpha
- 定位：新手标准型
- 攻击力：45 | 血量：60
- 弹道：双直线平行射击

### 苍雷-β
- 定位：高爆发穿透型
- 攻击力：70 | 血量：45
- 弹道：高能穿透光束

### 星穹-γ
- 定位：范围清场型
- 攻击力：40 | 血量：55
- 弹道：三向散射弹

### 天穹-Ω
- 定位：终极均衡强化型
- 攻击力：65 | 血量：75
- 弹道：锁定追踪弹

## 操作说明

### 键盘控制
- 方向键 / WASD：移动战机
- 空格键：射击
- P键：暂停/继续

### 触摸控制
- 触摸屏幕移动战机
- 自动射击

## 本地运行

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 部署到 Vercel

1. 将项目推送到 GitHub
2. 在 Vercel 导入项目
3. Vercel 会自动检测 Vite 项目并部署

## 技术栈

- React 18
- Vite
- Tailwind CSS 3
- HTML5 Canvas
- Font Awesome
